# Hey, I'm Andrés 👋

**DX Engineer @ [RealManage](https://www.realmanage.com/)** — Building developer platforms, cloud infrastructure, and the tooling that makes engineering teams move faster.

---

### 🛠️ What I Do

I design and maintain **cloud-native platforms** on AWS, automate everything that shouldn't be manual, and obsess over developer experience. My work sits at the intersection of infrastructure, security, and enablement.

- 🏗️ **Platform Engineering** — Multi-account AWS architectures, CI/CD pipelines, internal tooling
- 🔐 **Security & Governance** — IAM least-privilege, Identity Center, permission boundaries, compliance guardrails
- 🌐 **Networking** — Site-to-Site VPN, Transit Gateway, PrivateLink, hybrid connectivity
- 📦 **Infrastructure as Code** — Terraform, CloudFormation, CDK — modular, reusable, secure by default

### 🧩 Open Source — [Andrevops](https://github.com/Andrevops)

I build DX tooling in the open. These are VS Code extensions and utilities born from real workflow pain points:

  <tr>
    <td>
      <a href="https://github.com/Andrevops/diffchestrator"><b>🎛️ Diffchestrator</b></a><br/>

    </td>
    <td>
